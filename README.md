# Chess

This is my Object-Oriented Design chess project built in C# with Windows Forms.

The project supports:

- two-player mode
- single-player mode against a bot
- click-to-move and drag-and-drop movement
- legal move checking
- timers and turn display
- move history
- resign, new game, and home flow

## Project files

- main source code in the root project files
- images in `Piece Images`
- sound files in `Sounds`

## How to run

1. Open `Chess.sln` in Visual Studio.
2. Build the solution.
3. Run the `Chess` project.
